드래그 가능한 패널

// INSTALL

설치 방법

크롬 스토어 등록 없이 개발자 모드로 바로 설치할 수 있습니다.

1

ZIP 파일 다운로드 및 압축 해제

아래 다운로드 버튼을 클릭해 ZIP을 받고, 원하는 폴더에 압축을 풀어주세요.

2

크롬 확장 프로그램 페이지 열기

크롬 주소창에 chrome://extensions 를 입력하고 Enter.

3

개발자 모드 ON

페이지 우측 상단의 개발자 모드 토글을 켜주세요.

4

폴더 선택

압축해제된 확장 프로그램을 로드합니다 클릭 → 압축 해제한 폴더 선택.

5

플로우 메신저에서 사용

flow.team 또는 myflow.kr 접속 → 채팅방 / 프로젝트 열기 → 우측 하단 패널에서 ▶ 전체 백업 시작 클릭.

// NOTES

사용 전 알아두세요

🖱️ 백업 진행 중에는 채팅창을 건드리지 마세요. 스크롤이 끊기면 메시지가 누락될 수 있습니다.
🔧 플로우 업데이트로 DOM 구조가 변경되면 content.js 상단의 셀렉터를 수정해야 할 수 있습니다. F12 → Console에서 오류를 확인하세요.
🖼️ 이미지·첨부파일은 텍스트로 개수만 기록됩니다 (예: (이미지/첨부 2개)).
🔒 모든 처리는 브라우저 로컬에서만 이루어지며, 외부 서버로 데이터를 전송하지 않습니다.
